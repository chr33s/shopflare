# ShopFlare

> Shopify app running on cloudflare pages

## Requirements

1. Cloudflare account
2. cloudflared cli
3. Node.js & NPM see package.json#engines

## Setup

```sh
cp .env .env.local # update values

npm install
npm run tunnel
npm run start
```

## Deployment

```sh
npm run build
npm run deploy
```
